--> Project Overview

This repository contains Version 1 of the Novel Idea Bookstore website redesign, built using the Microsoft AI Stack (GitHub Copilot + VS Code). 
The site is modern, mobile-responsive, and completely static—no backend or database needed.

Novel Idea is a local independent bookstore with over 35 years of history in Kingston. This redesign brings their site into 2026 with clean 
design, easy navigation, and a professional online presence.

--> Tech Stack

HTML5: Clean semantic markup for better structure and accessibility  
CSS3: Custom color variables, gradients, Flexbox, and CSS Grid for responsive layouts  
JavaScript: Vanilla JS for mobile menu, form handling, and book filtering  
No Database Completely static site—works offline and loads instantly

--> How to Run

1. Clone or download this repository
2. Navigate to the `client-v1-Sailor` folder
3. Open `index.html` in any web browser.
4. Click links in the navigation to explore all pages

No installation, no server needed—just open and go.
